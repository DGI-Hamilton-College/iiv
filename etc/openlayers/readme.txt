Instructions for building OpenLayers.js file.

1. Get a copy of OpenLayers 2.7 (http://openlayers.org/)

2. copy the hcayless...\lib\OpenLayers\Layer\OpenURL.js file to 
    <OpenLayers>\lib\OpenLayers\Layers\OpenURL.js
    
3. copy the iiv.cfg file to <OpenLayers>\build

4. run <OpenLayers>\build\build.py iiv

5. copy <OpenLayers>\build\OpenLayers.js to <iiv>\WebContent\js\OpenLayers.js
